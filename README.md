Fargelegger App – React

En interaktiv React-app hvor brukeren kan fargelegge en rev tegnet i Figma. Hver del av SVG-en kan velges fra en dropdown eller ved å klikke direkte på delen, og fargen oppdateres umiddelbart.

Appen bruker useState for å holde på valgt del og farge, og props for å sende data til SVG-komponenten. Løsningen er responsiv, med modulære komponenter (ColorSelector og Figure).
